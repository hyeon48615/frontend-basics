연결 선택자(coimbination selector), 조합선택자

  • 연결 선택자 : 선택자와 선택자를 연결해 적용 대상을 제한하는 선택자
  • 하위 선택자(descendant selector)
    • 부모 요소에 포함된 ㅣ모든 하위 요소에 스타일을 적용
    • 자식 요소, 손자요소, 손자요소 등 모든 하위 요소에 반영
    • 하위 선택자 정의시에 상위 요소와 하위 요소를 나란히 작성
    • 기본형 상위요소 하위요소 ,,,
  • 자식 선택자(child selector)
    • 자식 요소 스타일을 적용하는 선택자
    • 두 요소 사이에 > (부등호) 를 사용하여 구분
    • 기본 요소 부모요소 > 자식요소
  • 인접형제 선택자(adjacent selector)
    • 같은 부모를 가진 형제 요소 중 첫번째 동생 요소에만 적용
    • 요소1 + 요소2
    • 요소1과 요소2는 같은 레벨이면서 요소1 이후 첫 번째 요소2에 적용
    • 기본형 요소1 + 요소2
  • 형제 선택자(sibling selector)
    • 형제 요소들에 스타일을 적용
    • 인접형제 선택자와 다른 점 : 모든 형제 요소에 적용됨
    • 기본형 요소1 ~ 요소2
  • 속성 선택자
    • 지정한 속성을 가진 요소를 찾아 스타일을 적용
    • [속성] 선택자
      • 해당 속성을 가진 요소에 모두 반영
    • [속성=값] 선택자
      • 특정 속성과 값을 가진 요소에 모두 반영
    • [속성~=값] 선택자
      • 여러 속성 값 중에 해당 값이 포함되어 있는 요소에 적용
    • [속성|=값] 선택자
      • 특정 값이 포함된 속성을 가지 요소에 스타일 적용
      • 하이픈으로 연결해 한 단어 값을 이루는 요소에 적용
    • [속성^=값] 선택자
      • 특정 값으로 시작하는 속성을 가진 요소에 적용
    • [속성$=값] 선택자
      • 특정 값으로 끝나는 속성을 가진 요소에 적용
    • [속성*=값] 선택자
      • 값의 일부가 일치하는 속성을 가지 요조에 적용
  • 가상 클래스 선택자
    • :link - 방문하지 않은 링크에 적용
    • :visited - 방문했던적이 있는 링크에 적용
    • :active - 웹 요소를 활성화했을 때 스타일 적용
    • :hover - 웹 요소에 마우스 커서를 올렸을 때
    • :focus - 웹 요소에 커서, 포인터가 맞추어졌을 때
    • 우선순위 :link -> :visited -> :hover -> :active
    • :target - 앵커로 연결된 부분에 스타일 적용
    • :enabled, :disabled - 요소의 사용 여부에 따라 스타일 적용
    • :checked - 라디오, 체크박스를 선택했을 때 적용
    • :not - 특정 요소를 제외하고 스타일 적용
  • 문서 구조에 따른 가상 클래스 선택자
    • :only-child - 부모 안의 자식 요소가 하나뿐일 떄 자식 요소 선택
    • A:only-type-of - 부모 안에 A 요소가 하나뿐일 때 선택
    • :first-child - 부모 안에 있는 모든 요소 중 첫 번째 자식 요소
    • :last-child - 부모 안에 있는 모든 요소 중 마지막 자식 요소
    • A:first-of-type - 부모 안에 있는 A 요소 중 첫번째 요소
    • A:last-of-type - 부모 안에 있는 A 요소 중 마지막 요소
    • :nth-child(n) - 부모 안에 있는 모든 요소 중 n번째 자식 요소
    • :nth-last-child(n) - 부모 안에 있는 모든 요소 중에서 끝에서 n번째 자식 요소
    • A:nth-of-type(n) - 부모 안에 있는 A 요소 중에서 n번째 요소
    • A:nth-last-of-type(n) - 부모 안에 있는 A 요소 중에서 끝에서 n번째 요소
  • 가상 요소
    • 화면 꾸미기용 요소를 웹 문서에 포함시키지 않기 위해 사용
    • ::first-line - 특정 요소의 첫 번째 줄에 적용
    • ::first-letter - 특정 요소의 첫 번째 글자에 적용
    • ::before - 특정 요소의 앞에 지정한 콘텐츠 추가
    • ::after - 특정 요소의 뒤에 지정한 콘텐츠 추가
