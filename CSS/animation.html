애니메이션

  • 웹 요소에 애니메이션 기능 추가
  • 애니메이션을 시작해 끝나는 동안 원하는 곳에서 스트일을 바꾸면서 애니메이션을 정의할 수 있음
  • 키프레임(keyframe) : 애니메이션 중간에 스타일이 바뀌는 지점점
  • @keyframes : 애니메이션이 바뀌는 지점
  • animation-delay : 애니메이션의 시작 시간을 지정
  • animation-direction : 종료된 뒤 처음부터 시작할지 역방향으로 진행할지 지정
  • animation-duration : 애니메이션의 실행 시간 지정
  • animation-iteration-count : 애니메이션의 반복 횟수 지정
  • animation-name : @keyframe으로 설정한 중간 상태 지정
  • animation-timing-function : 키프레임의 전환 형태를 지정
  • animation : animation 속성을 한번에 지정
  • @keyframes 속성
    • 애니메이션의 시작과 끝의 상태가 바뀌는 지점 설정
    • [이름] 으로 애니메이션을 구분
    • @keyframe 의 선택자에서 속성값이 바뀌는 지점을 가리킴
    • 시작 위치는 0%, 끝 위치는 100%로 놓고 위치 지정
    • 시작과 끝 위치만 사용하면 from, to 키워드를 사용 가능
    • 기본형 @keyframes 이름 { 선택자 { 스타일 } } @keyframes 이름 { 0% { CSS 코드 } n% { CSS 코드 } 100% { CSS 코드 } }
  • animation-name 속성
    • 어떤 애니매이션을 사용할지 구변
    • @keyframes 속성에서 만든 애니메이션 이름을 사용
    • 기본형 animation-name 키프레임이름 | none
  • animation-duration 속성
    • 애니메이션 실행시간 설정, 기본값 : 0
    • 사용 가능한 값 : 초, 밀리초
    • 애니메이션은 키프레임, animation-name, animation-duration 3가지를 필수로 지정해야 함
    • 기본형 animation-duration : 시간
  • animation-delay 속성
    • 애니메이션 지연 시간을 설정
    • 기본형 animation-delay : 지연시간간
  • animation-direction 속성
    • 애니메이션이 끝난 후 원래 위치로 돌아거나, 반대 방향으로 실행하도록 설정
    • 기본형 animation-direction: normal | alternate
    • normal : 끝까지 실행하면 원래 위치로 돌아감, 기본값
    • alternate : 끝까지 실행하면 되돌아가면서 다시 실행
    • reverse : 키프레임에 정의된 시간 역순으로 진행 (from ->to)
    • alternate-reverse : 애니메이션이 1회 이상 실행될 경우 홀수 번째는 reverse, 짝수 번째는 normal 로 진행
  • animation-iteration-count 속성
    • 애니메이션의 반복 횟수 지정
    • 기본형 animation-iteration-count : 숫자 | infinite
    • 숫자 : 반복 횟수
    • infinite : 무한 반복
  • animation-play-state 속성
    • 애니메이션의 진행/정지 상태를 정의의
    • 기본형 animation-play-state: 속성값
    • paused : 일시 정지
    • running : 실행
  • animation-timing-function 속성
    • 애니메이션 속도 속선을 지정
  • animation 속성
    • 여러 개의 animation 속성을 하나의 속성으로 줄여서 사용
    • 지정하지 않은 속성은 기본 값을 사용
    • animation-duration 값은 반드시 지정해야 함
    • 기본형 animation: 이름 duration timing-function delay iteration-count direction fill-mode play-state
    • animation : bgchange 5s 3 ease-in
  • animation-fill-mode 속성
    • 애니메이션의 시작과 끝을 비롯해 상태가 바뀌는 지점을 설정
    • 이름 으로 애니메이션을 구분
    • none : 실행전 - 시작시점(0%, from) 의 스타일을 적용하지 않고 대기 실행후 - 실행되기 전의 스타일 적용 상태로 변경
    • forward : 실행전 - 시작시점(0%, from) 의 스타일을 적용하지 않고 대기 실행후 - 키프레임에 정의된 종료시점(100%, to)의 스타일을 적용하고 대기
    • backwards : 실행전 - 시작시점(0%, from) 의 스타일을 적용하지 않고 대기 실행수 - 실행되기 전의 스타일 적용 상태로
    • both : 실행전 - 키프레임에 정의된 시작 시점(0%, from) 의 스타일 적용하고 대기 실행후 - 키프레임에 정의된 종료 시점(100%, to) 의 스타일 적용하고 대기
