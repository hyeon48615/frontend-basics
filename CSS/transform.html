Transform(변형, 트랜스폼)

  • 특정 요소의 크기나 형태 등의 스타일을 바꾸는 것
  • 2차원 변형
    • 수평이나 수직으로 웹 요소를 변형
    • 크기, 각도만 지정
    • 2차원 좌표 사용
    • translate(tx, ty) - 지정한 크기만큼 해당 축을 이동
    • translateX(tx) - x축으로 이동
    • translateY(ty) - y축으로 이동
    • scale(sx, sy) - 지정한 크기만큼 해당 축으로 확대, 축소
    • scaleX(sx) - 지정한 크기만큼 x축으로 확대, 축소
    • scaleY(sy) - 지정한 크기만큼 y축으로 확대, 축소
    • rotate(각도) - 지정한 각도만큼 회전
    • skew(ax, ay) - 지정한 각도만큼 해당 축으로 왜곡
    • skewX(ax) - 지정한 각도만큼 x축으로 왜곡
    • skewY(ay) - 지정한 각도만큼 y축으로 왜곡
  • 3차원 변형
    • x축과 y축에 원근감(z축) 추가
    • z축은 앞뒤로 이동, 보는 사람쪽으로 다가올수록 더 큰 값을 가짐
    • translate3d(tx, ty, tz) - 지정한 크기만큼 해당 축을 이동
    • translateZ(tz) - z축으로 이동
    • scale3d(sx, sy, sz) - 지정한 크기만큼 해당 축으로 확대, 축소
    • scaleZ(sz) - 지정한 크기만큼 z축으로 확대, 축소
    • rotate3d(각도) - 지정한 각도만큼 해당 축 기준으로 회전
    • rotateX(각도) - 지정한 각도만큼 해당 축 기준으로 회전
    • rotateY(각도) - 지정한 각도만큼 해당 축 기준으로 회전
    • rotateZ(각도) - 지정한 각도만큼 해당 축 기준으로 회전
    • perspective(길이) - 입체적으로 보일 수 있도록 깊이 값을 지정
  • transform-origin 속성
    • 변형 효과의 기준점을 변경할 때
    • 기본 형식 transform-origin : x축위치 y축위치
    • x축위치 : left / center / right
    • y축위치 : top / center / bottom
  • translate 함수
    • 지정한 방향으로 이동할 거리를 지정
    • 기본형 transform: 트랜스함수
    • transform: translate(tx, ty)
    • transform: translate3d(tx, ty, tz) - 지정한 크기만큼 이동
    • transform: translateX(tx) - x축으로 이동
    • transform: translateY(ty) - y축으로 이동
    • transform: translateZ(tz) - z축으로 이동
  • 트랜지션(전환, Transition)
    • 웹 요소의 스타일 속성이 조금씩 변하는 것
    • transition-property: 트랜지션의 대상을 지정
    • transition-duration: 실행할 시간을 지정
    • transition-timing-function: 실행 형태를 지정
    • transition-delay: 트랜지션의 지연 시간을 지정
    • transition: 위의 4가지 속성을 한번에 지정
    • trasition-property 속성
      • 트랜지션을 적용할 속성 선택
      • 지정하지 않으면 요소의 모든 속성이 대상이 됨
      • all : 속성값을 all 로 지정하거나 지정하지 않으면 모든 속성 대상
      • none : 트랜지션을 적용하는 동안 아무 속성도 바뀌지 않음
      • 속성이름 : 트랜지션을 적용하고 싶은 속성, 콤마(,)로 구분하여 여러 개 지정 가능
    • transition-duration
      • 트랜지션의 진행 시간을 지정
      • 시간단위 : 초, 밀리초
      • 여러개일 경우 , 로 구분하여 지정
      • 어떤 요소에 전환 효과를 지정하려면 반드시 transition-property 와 transition-duration 속성을 함께 지정해야 함
      • 기본형 trasition-duration: 시간 [,시간,,,]
    • transition-delay
      • 트랜지션이 언제부터 시작될지 지연 시간 설정
      • 시간단위 초, 밀리초, 기본값 : 0
      • 기본형 transition-delay: 시간
    • transition-timing-function
      • 트랜지션의 시작과 중간, 끝에서의 속도 지정
      • 기본형 transition-timing-function: 값
      • ease: 처음에는 천천히 시작하고 점점 빨라지다가 천천히 끝남, 기본값
      • linear : 시작부터 끝까지 같은 속도
      • ease-in : 느리게 시작
      • ease-out : 느리게 끝남
      • ease-in-out : 느리게 시작하고 느리게 끝남
      • cubic-bezier(n, n, n, n) : 배지에 함수를 정의하여 사용 n값 : 0~1 사이값만 사용
    • transition 속성
      • 트랜지션 관련 속성을 한꺼번에 지정
      • 시간 값 속성이 2개이므로, 앞에 오는 시간은 transition-duration, 뒤에 오는 시간은 transition-delay 속성
      • 기본형 transition: transition-property값 transition-duration값 transition-timing-function값 transition-delay값
