미디어쿼리

  • 미디어 쿼리의 기본 형식
    • 기본형 @media 미디어타입 and (추가 속성들 and | or | not ,,,,,) { CSS 코드 }
    • not : 뒤에 오는 모든 조건을 부정
    • only : 미디어 쿼리르 지원하는 기기만
    • 미디어타입(media type) : 미디어 쿼리가 적용될 미디어 종류
      • all : 모든 기기
      • print : 인쇄장치
      • screen : 컴퓨터, 스마트기기의 화면
      • speech : 스크린 리더기 같은 소리를 내어 읽어주는 장치
    • media feature : 미디어 쿼리가 적용될 미디어의 조건
      • min-width 화면너비 : 미디어 쿼리가 적용될 최소너비
      • max-width 화면너비 : 미디어 쿼리가 적용될 최대너비
      • orientation portrait : 세로모드, 뷰포트의 세로 높이가 가로 너비보다 큰 경우
      • orientation landscape : 가로모드, 뷰포트의 가로 너비가 세로 높이보다 큰 경우
  • 웹문서 기준
    • width, height : 웹페이지의 가로, 세로
    • min-width, min-height : 웹페이지의 최소 너비, 높이
    • max-width, max-height : 웹페이지의 최대 너비, 높이이
  • 단말기 기준
    • device-width, device-height : 단말기의 가로, 세로
    • min-deivce-width, min-device-height : 단말기의 최소 너비, 높이
    • max-deivce-width, max-device-height : 단말기의 최대대 너비, 높이
  • 회전
    • orientation : portrait - 단말기 세로모드
    • orientation : landscape - 단말기 가로모드
  • 화면 너비가 1440px 이상일 때 @media screen and (min-width: 1440px) { }
  • iphoneX 를 가로모드로 @media screen and(min-device-width: 375px) and (min-device-height:812px) { }
  • 미디어 쿼리의 중단점
    • 중단점(breakpoint) : 서로 다른 CSS 를 적용할 화면의 크기
    • 대부분 기기의 화면 크기 기준
    • 모바일, 태블릿, PC 기준으로 구분
    • PC 개발 -> 모바일 버전으로 개발 => 모바일 -> PC 개발
  • 미디어쿼리 적용
    • 1. 외부 css 파일 연결
      • 1. link 태그 사용
        • head 태그 내부에 정의
        • 기본형 <link rel="stylesheet" media="미디어쿼리조건" href="css파일경로">
        • <link rel="stylesheet" media="screen" href="./css/common.css">
      • 2. @import 구문 사용
        • style 구분 사이에 삽입
        • 기본형 @import url(css파일경로) 미디어쿼리조건설정
        • @import url("./css/table.css") only screen and (min-width:321px) and (max-width:768px);
    • 2. 웹 문서에 직접 정의
      • 1. style 태그 안에서 media 속성 정의
        • 기본형 <style media="조건"> { CSS 작성 }</style>
      • 2. @media 문 사용
        • 기본형 <style> @media 조건 { CSS 작성 } </style>
  • 레이아웃
    • 그리드 레이아웃
      • 반응형 웹 디자인에서 웹 문서 요소를 배치하는 기준
      • 웹사이트 화면을 여러 개의 컬럼으로 나눈 후 웹 요소를 배치
      • 화면을 규칙적으로 일관성 있게 유지할 수 있음음
      • 시각적으로 안정된 디자인
      • 업데이트가 편한 웹 디자인 구성
      • 요소를 자유롭게 배치
    • 레이아웃 작성 방법
      • 1. 플렉스박스 레이아웃
        • 수평 방향이나 수직 방향 중 하나를 주축으로 정하고 박스를 배치
        • 여유 공간이 생길 경우 너비나 높이를 적절하게 늘리거나 줄일 수 있음음
        • 플렉스 컨테이너(부모박스) : 플렉스 박스 레이아웃을 적용할 대상을 묶는 역할
        • 플렉스 항목/아이템(자식박스) : 플렉스 박스 레이아웃을 적용할 대상
        • 주축(main axis) : 플렉스 컨테이너 안에서 플렉스 항목을 배치하는 기준 , 기본값 왼쪽->오른쪽 , 플렉스 항목이 배치가 시작되는 부분 -> 주축시작점 , 항목 배치가 끝나는 지점 -> 주축끝점
        • 교차축(cross axis): 주축과 교차하는 방향, 기본값 위에서 아래로 배치 , 플렉스 항목의 배치가 시작되는 부분 -> 교차축 시작점 , 배치가 끝나는 부분 -> 교차축 끝점
        • display 속성
          • 배치 요소를 감싸는 부모 요소를 플렉스 컨테이너로 지정
          • flex : 컨테이너 안의 플렉스 항목을 블록 레벨로 요소로 배치
          • inline-flex : 컨테이너 안의 플렉스 항목을 인라인 레벨 요소로 배치
        • felx-direction 속성
          • 플렉스 항목의 배치를 위해 주축, 방향을 지정
          • row : 주축을 가로로 지정, 왼쪽에서 오른쪽으로 배치, 기본값
          • row-reverse : 주축을 가로로 지정, 오른쪽->왼쪽으로 배치
          • column : 주축을 세로로 지정, 위쪽->아래래쪽으로 배치
          • column-reverse : 주축을 세로로 지정, 아래쪽->위쪽으로 배치치
        • flex-wrap 속성
          • 플레스 항목을 한 줄 또는 여러 줄로 배치
          • nowrap : 한 줄에 표시, 기본값
          • wrap : 여러 줄에 표시
          • wrap-reverse : 여러 줄에 표시, 시작점과 끝점이 바뀜
        • flex-flow 속성
          • 플렉스 배칭 방향과 여러 줄 배치를 한꺼번에 지정
          • 기본형 flex-flow : row no-wrap
        • justify-content 속성
          • 플레스 항목을 주축 방향으로 배치할 때 배치 기준
          • flex-start : 주축의 시작점에 맞춰서 배치
          • flex-end : 주축의 끝점에 맞춰 배치
          • center : 주축의 중앙에 맞춰 배치
          • space-between : 첫 번째 항목을 주축 시작에 맞추고, 끝 항목을 주축의 끝에 맞춘후 , 각 항목 사이의 간격 동일하게 배치
          • space-around : 모든 항목을 주축에 같은 간격으로 배치
          • space-evenly : 플레스 아이템 사이와 양끝의 간격이 균일하게 배치
        • align-item 속성, align-self 속성
          • 교차축을 기준으로 하는 배치 방법을 조절
          • 교차축에서 특정 항목만 지정하고 싶을 때 align-self 를 사용 -> 해당 항목 요소에 사용
          • align-content 속성 : 플렉스 아이템이 두 줄 이상일때 교차축 방향으로 정렬
          • align-self 속성 : 각각의 플레스 아이템을 교차축 방향으로 정렬 , align-items 속성으로 플렉스 아이템을 한 번에 정렬하지 않고 각각 정렬하고 싶을 때 사용
          • flex-start : 교차축의 시작점에 맞춰서 배치
          • flex-end : 교차축의 끝점에 맞춰서 배치
          • center : 교차축의 중앙에 배치
          • baseline : 교차축의 문자 기준선에 맞춰 배치
          • stretch : 플렉스 항목을 늘려서 교차축에 가득차게 배치
        • align-content 속성
          • 플레스 항목이 여러 줄로 표시될 때 교차축 기준의 배치 방법 설정
          • flex-start : 교차축 시작점에 맞춰 배치
          • flex-end : 교차축의 끝점에 맞춰서 배치
          • center : 교차축의 중앙에 배치
          • space-between : 첫 번째 항목을 교차축 시작점과 끝점에 맞추고, 각 항목 사이의 간격 동일하게 배치
          • space-around : 모든 항목을 교차축에 같은 간격으로 배치
          • stretch : 플렉스 항목을 늘려서 교차축에 가득차게 배치
      • 2. CSS 그리드 레이아웃
        • 최근에 등장한 CSS 표준
        • 수평 방향이나 수직 방향 어디로든 배치 가능
        • 레고 블록 끼우듯 요소를 배치할 수 있음
        • 레이아웃 양쪽 방향을 모두 사용
        • 줄(row), 컬럼(column) 으로 화면을 구성, 줄 사이의 여백, 컬럼 사이의 여백을 조절
        • 행(row) : 그리드 레이아웃의 가로줄
        • 열(column) : 그리드 레이아웃의 세로줄
        • 그리드셀(grid cell) : 행과 열이 만나서 이루는 사각형
        • 그리드갭(grid gap) : 셀과 셀사이의 공간
        • 그리드아이템(grid item) : 그리드 셀 안에 포함되는 컨텐츠
        • 그리드라인(grid line) : 그리드 행과 열을 그리는 선
        • 그리드넘버(grid number) : 그리드 라인에 붙이는 번호
        • 그리드컨테이너(grid cotainer) : 그리드 아이템을 묶는 부모 요소
        • display 속성
          • 배치 요소를 감싸는 요소를 그리드 컨테이너로 지정
          • grid : 컨테이너 안의 항목을 블럭레벨 요소로 배치
          • inline-grid : 인라인 레벨 요소로 배치
        • grid-template-columns, grid-template-rows 속성
          • grid-template-columns : 그리드 컨테이너 안의 컬럼 개수, 너비 값
          • grid-template-rows : 그리드 컨테이너 안의 줄의 개수, 너비값
          • 기본형 grid-template-columns : 1열값, 2열값,,,,,
          • 기본형 grid-tempalte-rows : 1행값, 2행값,,,,,
          • row-gap, column-gap : 그리드 셀, 셀 사이의 간격 지정
          • 기본형 row-gap : 크기, column-gap : 크기
        • 그리드 레이아웃 정렬 속성
          • align-items 속성
            • 그리드 아이템 전체를 셀의 세로 방향으로 정렬
            • stretch : 그리드 아이템과 셀을 채우도록 크기를 늘림
            • start : 그리드 아이템을 그리드 셀의 맨 위에 배치
            • center : 그리드 아이템을 그리드 셀의 세로 방향 중아에 배치
            • end : 그리드 아이템을 그리드 셀의 맨 아래에 배치
          • align-self 속성
            • 각각의 그리드 아이템을 셀의 세로 방향으로 정렬
          • place-item 속성
            • align-items, justify-items 속성을 한 번에 적용
            • 기본형 place-items : align-items, justify-items
            • 예) place-items : center, end
          • place-slef 속성
            • align-self, justify-self 속성을 한 번에 적용
            • 기본형 place-self : align-self, justify-self
            • 예) place-self : center end
          • 상대적인 크기 단위
            • fr 단위 : 컬럼/줄의 크기 지정시 px 단위는 반응형 웹 디자인에 적합치않음 -> 상대적인 크기 지정하는 fr(fraction) 단위 사용
            • 예) 너비 비율 2:1:2 인 컬럼 3개 배치 -> grid-template-columns: 2fr 1fr 2fr;
            • repeat() 함수 -> 똑같은 내용이 반복될 때 사용
            • 예) grid-template-columns:repeat(3, 1fr)
          • 컬럼 갯수 조절
            • 컬럼 너비 설정시에 auto-fit, auto-fill 이용하면 컬럼 개수를 조절할 수 있음
            • auto-fit : 남는 공간 없이 꽉 채우기
            • auto-fill : 컬럼의 최소 너비까지만 표시하고 남는 공간은 그대로 둠
            • 예) 너비 200px 인 컬럼을 화면에 가득 채우기 -> grid-template-columns:repeat(auto-fit, 200px);
        • 그리드 레이아웃의 배치 속성
          • grid-tempalte-areas 속성
            • 그리드 레이아웃에서 행과 열을 이름으로 지정
            • grid-area 속성 : 그리드 아이템에 이름을 지정
            • 기본형 grid-area : 행, 열 이름으로
          • grid-column-start, grid-column-end 속성
            • 그리드 레이아웃에서 열의 시작 번호와 끝번호 지정
          • grid-row-start, grid-row-end 속성
            • 그리드 레이아웃에서 행의 시작과 끝 번호 지정
          • 그리드 라인 : 그리드 컨테이너를 구성하는 행과 열을 그리는 선
          • 그리드 넘버 : 그리드 라인에 있는 고유 번호
          • grid-column 속성 : grid-column-start, grid-column-end 속성을 한 번에 적용
            • 기본형 grid-column: start end
            • 기본형 grid-column: start/span 열개수
          • grid-row 속성 : grid-row-start, grid-row-end 속성을 한 번에 적용
            • grid-row : start end
            • grid-row : start/span 행개수
