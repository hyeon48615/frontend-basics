CSS 박스 모델

  • 블록 레벨 요소
    • 요소를 삽입했을 때 한 줄을 차지하는 요소
    • 요소의 너비가 100%
    • div, p 등등
  • 인라인(넌블럭) 요소
    • 줄을 차지하지 않는 요소
    • 컨텐츠의 길이만큼 화면 영역 차지하고, 나머지 공간에 다른 요소가 올 수 있음
    • img, strong, span, label 등
  • width, height 속성
    • width : 실제 컨텐츠 영역의 너비 크기를 지정
    • height : 실제 컨텐츠 영역의 높이 크기를 지정
    • auto : 컨텐츠의 크기에 따라 자동으로 결정, 기본값
  • box-sizing 속성
    • 실제 박스 모델의 너비 계산시 포함하는 영역 결정하는 속성
    • border-box : 테두리까지 포함하여 너비 값 계산
    • content-box : 컨텐츠 영역만 너비 값으로 계산, 기본값값
  • box-shadow 속성
    • 선택한 요소에 그림자 효과를 냄냄
    • 수평 거리 : 그림자가 가로로 얼마나 떨어져 있는가, 양수 : 오른쪽, 음수:왼쪽
    • 수직 거리 : 그림자가 세세로로 얼마나 떨어져 있는가, 양수 : 아래쪽, 음수: 위쪽
    • 흐림정도 : 그림자의 진한 정도, 0이 기본값, 음수 사용 불가
    • 번짐정도 : 양수 - 모든 방양으로 퍼져서 표시, 음수 - 모든 방향으로 그림자가 축소되어 표시시
    • 색상 : 한가지 색상 지정, 여러가지 색상 지정 가능, 기본값:검정색
    • inset : 안쪽 그림자로 표시
    • 기본형 box-shadow : 수평거리 수직거리 흐림정도 번짐정도 생상 inset
  • box 모델의 값 설정
    • margin, padding 값의 지정 방향 존재
    • 전체 방향 지정시에는 방향성이 없이 하나의 값으로 설정
    • top -> right -> bottom -> left
    • margin : 10px
    • margin-top : 10px
    • margin-right : 10px
    • margin-bottom : 10px
    • margin-left : 10px
    • margin : 10px 10px 10px 10px
    • margin : 10px 20px
  • border-style 속성
    • 기본값 : none -> 테두리 표시 안됨
    • 테두리를 그리기 위해서는 테두리 스타일 지정해야 함
    • 기본형 boder-style : none | hidden | dashed | dotted | solid | double | groove | inset | outset | ridge
    • none : 테두리 없음
    • hidden : 테두리를 표시하지 않음, collapse 일 경우에도 표기하지 않음
    • solid : 실선
    • dotted : 점선
    • dashed : 파선(짧은 점선)
    • double : 이중선
    • groove : 테두리를 창에 조각한 것처럼 표시
    • inset : 표에서 border-collapse:separate 일 경우 전체 테두리를 창문처럼 표시, collapse 일 경우 groove 와 동일
    • outset : 표에서 border-collapse:separate 일 경우 전체 테두리가 튀어나온것처럼 표시, collapse 일 경우 ridge 와 동일
    • ridge : 테두리가 창틀에서 튀어나온 것처럼 표시
  • border-width 속성
    • 테두리의 두께 지정
    • 기본형 border-width : 크기 | thin | medium | thick
    • border-width : top right bottom left
    • border-width : 값
  • border-color 속성
    • 테두리의 색상 지정
    • border-color : 값 -> 4방향 모두
    • border-top-color, -right-color, -bottom-color, -left-color -> 각각 해당 방향의 선색 지정
  • boder-radius 속성
    • 박스 모델의 테두리의 꼭지점 부분을 둥글게 처리
    • 기본형 border-radius : 크기 | 백분율
    • 크기 단위 :ps, em 을 사용하여 반지름의 크기를 수치로 표시
    • 백분율 : 현재 요소의 크기 기준으로 비율(%)를 지정
    • border-top-left, border-top-right, -bottom-right, -bottom-left
  • display 속성
    • 요소의 배치 방법을 결정
    • HTML 태그가 기본적으로 가지고 있는 박스 모델의 성격을 변경
    • 박스 모델의 성경
      • 블록성격 : 요소의 너비가 컨텐츠의 유무와 상관없이 항상 가로 한 줄을 차지
      • 인라인(논블럭) 성격 : 요소의 너비를 컨텐츠 크기만큼만 차지
      • 인라인 블록 : 요소의 너비가 컨텐츠의 크기만큼 차지하지만 그외 성격은 블록 성격을 가지는 것
      • block : 인라인 레벨 요소를 블록 레벨 요소로 만듦
      • inline : 블록 레벨 요소를 인라인 레벨 요소로 만듦
      • inline-block : 인라인 레벨 요소와 블록 레벨 요소의 속성을 모두 가지고 있으며, 마진과 패딩 등을 지정할 수 있음
      • none : 해당 요소를 화면에 표시하지 않음
  • float 속성
    • 요소를 왼쪽, 오른쪽 등으로 위치
    • 적용대상의 원래 위치를 보장하지 않음
    • 기본형 float: left | right | none
    • left : 해당 요소를 문서의 왼쪽으로 배치
  • clear 속성
    • float 속성을 무효화 시키는 속성
    • 기본형 clear : none | left | right | both
    • left : float:left 해제
    • right : float:right 해제
    • both : left, right 를 해제
  • z-index 속성
    • position 속성으로 배치한 요소의 z축 위치를 결정
    • 형식 z-index:정수값
    • 1~1000
  • 웹 요소의 위치 지정
    • left : 기준 위치와 요소 사이의 왼쪽의 공백
    • right : 기준 위치와 요소 사이의 오른쪽 공백
    • top : 기준 위치와 요소 사이의 위쪽 공백
    • bottom : 기준 위치와 요소 사이의 아랫쪽 공백백
  • position 속성
    • 웹 문서에 요소를 배치하기 위한 속성
    • 기본형 position: static | relative | absolute | fixed
    • static : 요소를 기본 흐름에 따라서 배치
    • relative : 요소를 기본 흐름에 따라서 배치, 좌표 속성 사용
    • absolute : 기본 흐름에서 벗어나서 절대적인 좌표 위치에 표시
    • fixed : 지정한 위치에 고정하여 배치, 화면에서 잘리거나, 보이지 않을 수 있음
