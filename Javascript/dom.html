문서 객체 모델(DOM, Document Object Model)

  • 자바스크립트를 이용하여 웹 문서에 접근하고 제어할 수 있도록 객체를 쳬계적으로 관리할 수 있도록 정리한 방법
  • 웹 문서와 그 안의 요소를 개체로 인식하고 처리
  • 돔 트리(Dom Tree)
    • 웹 문서의 요소들 간의 관계를 계층 구조로 표시한 것
    • 나무 형태가 되기 때문에 DOM 트리 라고 함
    • 노드(node) : DOM 트리에서 가지가 갈려져 나온 항목
    • 루트 노드(root node) : DOM 트리의 시작 부분(html)
  • DOM 구성하는 원칙
    • 모든 HTML 태그는 요소(element) 노드
    • 웹 문섯의 텍스트 내용은 요소 노드의 자식 노드 -> 텍스트(text) 노드
    • 태그의 속성은 요소 노드의 자식 노드인 속성(attribute) 노드
    • 주석은 주석(comment) 노드
  • 노드 타입
    • 문서 노드(Node.DOCUMENT_NODE) : 최상위 document 객체의 노드 타입
    • 요소 노드(Node.ELEMENT_NODE) : 요소의 노드 타입 예) h1, p, div 등 태그
    • 속성 노드(Node.ATTRIBUTE_NODE) : 속성의 노드 타입 예) href, src 속성 등
    • 텍스트 노드(Node.TEXT_NODE) : 텍스트에 해당하는 부분
    • 주석 노드(Node.COMMENT_NODE) : 주석에 해당하는 부분
  • 속성으로 노드 선택
    • 모든 요소 탐색 속성
      • parentNode : 부모 노드를 반환
      • childNodes : 모든 자식 노드 반환
      • firstChild : 첫 번째 자식 노드 반환
      • lastChild : 마지막 자식 노드 반환
      • previousSibling : 이전 형제 노드 반환
      • nextSibling : 다음 형제 노드 반환
    • 요소 노드 탐색 속성
      • parentElement : 부모 요소 노드 반환
      • children : 자식 요소 노드 반환
      • firstElementChild : 첫 번째 자식 요소 노드 반환
      • lastElementChild : 마지막 자식 요소 노드 반환
      • previousElementSibling : 이전 요소 노드 반환
      • nextElementSibling : 다음 요소 노드 반환
  • 메서드로 노드 선택
    • 속성값과 태그명 사용
      • getElementById('id 속성값') : id 속성값이 일치하는 요소 노드 1개 선택
      • getElementsByClassName('class 속성값') : class 속성값이 일치하는 요소 노드 모두 선택
      • getElementsByTagName('태그명') : 태그명과 일치하는 요소 노드 모두 선택
    • CSS 선택자 사용
      • querySelector('CSS 선택자') : CSS 선택자에 해당하는 노드 1개 선택
      • querySelectorAll('CSS 선택자') : CSS 선택자에 해당하는 노드 모두 선택
  • 노드 조작하기
    • 컨텐츠 조작
      • textContent : 노드 요소의 모든 텍스트 요소 접근에 사용
      • innerText : 노드 요소의 텍스트 중 웹 브라우저에 표시되는 텍스트만 접근
      • innerHTML : 노드 요소의 텍스트 중 HTML 태그를 포함한 텍스트만 접근
    • 스타일 조작
      • 기본형 노드.style.
      • 기본형 노드.style.=<속성값>
    • 클래스 속성 조작
      • 기본형 <노드>classList.add("class 속성값") // 추가
      • 기본형 <노드>classList.remove("class 속성값") // 삭제
      • 기본형 <노드>classList.toggle("class 속성값") // 추가/삭제 반복
    • 데이터 속성 조작
      • 사용자 정의(custom) 속성
      • dataset 속성을 이용하여 조작
      • dataset 은 HTML 문서상의 data-* 속성을 가져올 때 사용
    • 메서드 속성 조작
      • 기본형 <노드>.getAttribute("속성명"); // 속성 값 조회
      • 기본형 <노드>.setAttribute("속성명", "속성값"); // 속성 값 설정
      • 기본형 <노드>.removeAttribute("속성명"); // 속성 삭제
    • 문자열 처리
      • bold() : 문자열 b 태그 적용
      • charAt(인덱스) : 인덱스 부분에 있는 문자를 리턴
      • concat(문자열2, 문자열3) : 문자열1.concat(문자열2, 문자열3) : 문자열1 + 문자열2 + 문자열3
      • fontcolor() : fontclor(색상) -> 폰트색상 지정
      • lastIndexOf(문자열) : 문자열이 시작되는 인덱스 값 리턴
      • 문자열1.replace(문자열2, 문자열3) : 문자열1에서 문자열2를 문자열3으로 바꿈, 처음 만나는 문자열 하나만 바꿈
      • 문자열1.replaceAll(문자열2, 문자열3) : 문자열1에서 문자열2를 문자열3 으로 모두 바꿈
      • 문자열1.search(문자열2) : 문자열1에서 문자열2가 검색되었을 경우 시작인덱스를 리턴
      • 문자열1.include(문자열2) : 문자열1에 문자열2가 존재하면 true, 아니면 false 리턴
      • 문자열1.slice(인덱스1, 인덱스2) : 문자열1을 인덱스1 ~ 인덱스2까지 자른 문자열 리턴
      • 문자열1.substring(시작인덱스[, 종료인덱스]) : 시작인덱스 ~ 끝까지, 시작인덱스 ~ 종료인덱스-1 까지 문자열 리턴
      • 문자열1.split(문자열2) : 문자열2를 기준으로 문자열1을 나눠서 array 로 리턴
      • 문자열1.toUpperCase(), .toLowerCase() : 대소문자 변환
      • 문자열.trim() : 문자열 앞뒤 공백 제거
      • 문자열.trimStart() : 문자열 앞 공백 제거
      • 문자열.trimEnd() : 문자열 뒤 공백 제거
      • 문자열.valueOf() : 문자열 객체, array 일 때 결과값을 문자열로 리턴
      • 문자열.length : 문자열의 길이값 리턴 (속성. 메서드X)
  • 노드 추가/삭제
    • 요소 노드 생성
      • 기본형 document.createElement("추가할 요소")
    • 텍스트 노드 생성
      • 기본형 document.createTextNode("추가할 텍스트 문구")
    • 자식 노드 연결
      • 기본형 부모노드.appendChild(자식노드)
      • 텍스트 노드 예) 부모노드 pp pp.apeendChild(txtNode); document.getElementBy('부모노드').appendChild(자식노드);
    • 속성 노드 추가
      • 기본형 document.createAttribute("속성")
      • 이미지 src 속성 추가 예) document.createAttribute("src").value = "이미지 경로";
    • 생성한 속성을 노드에 연결
      • 기본형 요소이름.setAttribute(속성노드)
      • 이미지노드.setAttribute(속성)
