기본 문법

  • 기본용어
    • 변수(vliable) : 값이 달라질 수 있는 데이터를 담는 공간
    • 상수(constant) : 값을 한 번 지정하면 바뀌지 않는 정적 데이터를 담는 공간
    • 키워드(keyword) : 자바스크립트에서 어떤 역할이나 기능이 정해진 단어. 예약어(reserved word)라고도 함
    • 식별자(identifier) : 자바스크립트 내부에 변수, 함수 등에 부여되는 이름
    • 연산자(operator) : 연산 작업에 사용되는 기호
    • 표현식(expression) : 평가되어 (계산되어) 나오는 하나의 값 혹은 반환식, 코드
    • 값(value) : 더 이상 평가할 수 없는 데이터
    • 세미콜론(semicolon) : 하나의 문(statement, 문법)이 끝났음을 의미
  • 변수 선언 규칙
    • 변수 이름
      • 영문자, 언더스코어(_), $, -, 숫자 사용
      • 첫 글자 영문자, _기호, $기호만 올 수 있음
      • 공백 허용하지 않음
      • 예약어(키워드)를 사용할 수 없음
      • 여러 단어로 이루어진 변수를 사용할 경우, _, - 단어를 직접 연결하여 사용할 수 있음
    • 변수 선언
      • var 키워드 사용
      • var 한 번만 쓰고 변수명은 , 이용하여 여러 개 선언 가능
    • 변수에 값을 할당
      • 변수명 = 값; 할당 후 변수명으로 처리
      • var 변수명; 변수명=값;
      • var 변수명 = 값;
      • 관용적 식별자 명명 규칙
        • 식별자는 영문으로만 작성, 예) name, age
        • 식별자는 의미있는 단어로 구성
      • 식별자 표기법
        • 카멜 표기법 : 변수명과 함수명 작성 시 주로 사용, 첫 시작문자 소문자 + 중간 단어 시작은 대문자 예) firstName, lastName
        • 언더스코어 표기법 : 주로 상수명 작성 시 사용, 주로 전체 단어를 대문자로 표현 예) PI, ADULT_AGE : 18
        • 파스칼 표기법 : 생성자 함수, 클래스 작성 시 사용 예) FirstName, LastName
  • 자료형
    • 컴퓨터가 처리할 수 있는 자료의 형태
    • 기본유형
      • 숫자형
        • 따옴표 없이 숫자로만 표기
        • 예) var birthYear = 2004;
        • 정수 : 소수점이 없는 숫자
        • 실수 : 소수점이 있는 숫자 -> 자바스크립트는 정밀도가 정확하지 않으므로 주의하여야 함
      • 문자열
        • 작음 따옴표('), 큰따옴표(")로 묶어서 나타냄
        • 숫자로 묶으면 문자로 인식
        • 에) var year = 2004; / var year = "2004";
        • 문자열 안에 ', " 존재할 경우 작은 따옴표 -> 큰따옴표로 감싸서("'") , 큰 따옴표 -> 작은 따옴표로 감싸서('"') , 이스케이프문자 사용 -> (\"), (\')
        • 템플릿 문자열 : 백틱(`)을 이용하여 여러 줄의 문자열을 묶어서 표현하던가, 문자열 안에 변수 등을 표현
      • 논리형
        • 참(true), 거짓(false) 2가지 값만 갖는 유형
        • true, false 소문자로만 표기
        • 예) var isFact = true;
    • 복합유형
      • 배열
        • 하나의 변수에 여러 개의 값을 지정
        • 각각의 요소에 접근 : 인덱스 이용
        • 예) var season = ['봄', '여름', '가을', '겨울'];
        • season[0] -> 봄 을 나타냄
      • 객체
        • 함수와 속성을 함께 포함
        • 예) var date = new Date();
        • date.now();
        • 배열, 함수, 객체 리터럴 등으로 파생되는 자료형
        • 객체리터럴 : 중괄호({})를 사용, 키(key)와 값(value)의 한쌍으로 이루어진 자료 구조
        • 기본형 { 키1: 값1, 키2: 값2,,,, }
    • 특수유형
      • undefined
        • 자료형이 지정되지 않았을 때의 상태
        • 변수만 선언하고 값을 할당하지 않은 변수의 상태
        • 리턴값 : undefined
      • null
        • 값이 유효하지 않을 때의 상태
    • 연산자
      • 산술연산자
        • + : 덧셈 연산자
        • - : 뺄셈
        • * : 곱셈
        • / : 나눗셈
        • % : 나머지 연산자(모듈러 연산자)
        • ++ : 피연산자를 1 증가시킴
        • -- : 피연산자를 1 감소시킴
        • ** : 거듭제곱
        • y++ : 후치연산자
        • ++y : 전치연산자
      • 할당 연산자(대입연산자)
        • 연산자를 기준으로 오른쪽의 결과를 왼쪽 변수에 할당
        • = -> 연산자의 오른쪽 결과를 왼쪽 변수에 할당
        • += -> 오른쪽 값을 더한 후 왼쪽에 할당
        • -= -> 오른쪽 값을 뺀 후 왼쪽에 할당
        • *= -> 곱하기
        • /= -> 나누기
        • %= -> 나머지
        • **= -> 제곱
      • 연결 연산자
        • 둘 이상의 문자열을 합쳐서 새로운 문자열을 만드는 연산자
      • 비교 연산자
        • 피연산 2개의 값을 비교하여 true 혹은 false 를 리턴
        • == -> 피연산자의 값이 같을 때 true
        • === -> 피연산자의 값과 자료형이 같을 때 true
        • != -> 피연산자의 값이 같지않으면 true
        • !== -> 피연산자의 값과 자료형이 같지 않으면 true
        • < -> 왼쪽 값이 작으면 true
        • <= -> 왼쪽 값이 작거나 같으면 true
        • > -> 왼쪽 값이 크면 true
        • >= -> 왼쪽 값이 크거나 같으면 true
      • 논리 연산자
        • true 와 false 가 피연산자인 연산 조건을 처리할 때 사용
        • OR 연산자 ( || ) -> 피연산자 중 하나만 true 여도 true -> | 연산자와의 차이점 ?
        • AND 연산자 ( && ) -> 피연산자 모두 true 일 때만 true -> & 연산자와의 차이점 ?
        • NOT 연산자 ( ! ) -> 피연산자의 반대값을 지정
