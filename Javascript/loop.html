반복문

  • for문
    • 기본형 for(초기값; 조건; 증가값) { 실행문; }
    • 초기값 : 카운터 변수를 초기화
    • 조건 : 명령을 반복하기 위한 조건 체크
    • 증가값 : 명령의 반복 수행. 보통 1씩 증가
    • 중첩된 for문 기본형 for(초기값; 조건; 증가값) { for(초기값; 조건; 증가값) { 실행문; } }
    • for~in 구문
    • for-of 구문
  • while문
    • 조건을 체크하고 true 면 {} 의 구문 실행
    • 조건 false 면 {} 은 실행되지 않음
    • 기본형 whild (조건) { 실행구문; }
  • do ~ whild문
    • 일단 명령을 한 번 실행 후 조건 체크
    • 조건이 true 면 {} 안의 구문 실행
    • 조건이 false 여도 {} 구문은 최소 한 번은 실행
    • 기본형 do { } while (조건);
  • break
    • 종료 조건이 되기 전에 반복문을 빠져나와야 할 때 사용
    • 기본형 break;
  • continue
    • 조건에 해당하는 값을 만나면 반복문의 맨 앞으로 이동 -> 반복 과정을 한 차례 건너뛰는 효과
    • 기본형 continue;
