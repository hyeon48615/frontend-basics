브라우저 객체

  • window : 브라우저 창이 열릴 때마다 생성. 브라우저에서 최상위 요소
  • document : 웹 문서마다 하나씩 존재. body 태그 생성시 생성. HTML문서의 루트 요소
  • navigator : 현재 사용하는 브라우저 정보 포함
  • history : 현재 창에서 사용자의 방문 기록을 저장
  • location : 현재 페이지의 URL 정보 포함
  • screen : 현재 사용하는 화면 정보
  • window 객체의 프로퍼티
    • document : 브라우저 창에 표시되는 웹문서에 접근
    • frameElement : 현재 창이 다른 요소 안에 포함되어 있을 경우 객체를 반환, 포함되어 있지 않으면 null 반환
    • innerHeight : 내용 영역의 높이
    • innerWidth : 내용 영역의 너비
    • localStrage : 웹브라우저에 데이터 저장하는 영역
    • location : window 객체의 위치, 현재 URL을 표시
    • name : 브라우저 창의 이름 조회, 수정
    • outerHeight : 브라우저 창의 바깥 높이
    • outerWidth : 브라우저 창의 바깥너비
    • pageXOffset : 스크롤했을 때 수평으로 이동하는 픽셀 수, scrollX와 동일
    • pageYOffset : 스크롤했을 때 수직으로 이동하는 픽셀 수, scrollY와 동일
    • parent : 현재 창이나 서브 프레임의 부모
    • screenX : 브라우저 창의 왼쪽 테두리가 모니터 왼쪽 테두리에서 떨어져 있는 거리
    • screenY : 브라우저 창의 위쪽 테두리가 모니터 위쪽 테두리에서 떨어져 있는 거리
    • scrollX : 스크롤했을 때 수평으로 이동하는 픽셀 수 -> pageXoffset
    • scrollY : 스크롤했을 때 수직으로 이동하는 픽셀 수 -> pageYoffset
    • sessionStorage : 브라우저에서 데이터를 저장하는 세션 스토리지 반환
    • screenTop : 웹 브라우저의 위쪽과 모니터 사이의 간격
    • screenLeft : 웹 브라우저의 왼쪽과 모니터 사이의 간격
  • navigator 객체
    • 브라우저 종류에 따라 속성 값이 다를 수 있음
    • battery : 배터리 충전 상태
    • cookieEnabled : 쿠키 정보 사용 여부 - true: 사용, false: 사용안함
    • geolocation : 모바일 기기를 이용한 위치 정보
    • language : 브라우저 UI 의 언어 정보
    • oscpu : 운영체제 정보
    • userAgent : 현재 브라우저 정보를 담고 있는 에이전트 정보
  • history 객체
    • 방문한 사이트 주소가 배열 형태로 저장
    • length : 현재 브라우저 창의 history 목록에 있는 항목 수
    • back : history 목록에서 현재 페이지 기준 이전 페이지를 현재 화면으로 이동
    • forward : history 목록에서 현재 페이지 기준 다음 페이지를 현재 화면으로 이동
    • go : history 목록에서 현재페이지 기준 이동할 페이지 지정, 이전 -> -, 다음 -> +, history.go(-1); -> 이전 페이지, history.go(1); -> 다음 페이지
  • location 객체
    • 현재 문서의 URL 주소 정보가 담겨 있음
    • 프로퍼티
      • hash : URL 중에서 #으로 시작하는 해시 부분의 정보
      • host : URL 에서 호스트이름 + 포트번호
      • hostname : URL 의 호스트 이름
      • pathname : URL 의 경로
      • port : URL 의 포트번호
      • protocol : URL 의 서비스 프로토콜 정보
      • password : 도메인 이름 앞에 username 과 password 를 함께 입력해서 접속하는 사이트일 경우 보유
      • search : URL 중에서 ? 시작하는 검색 내용
      • username : 도메인 이름 앞에 username 을 함께 서서 접속하는 사이트
    • 메서드
      • assign : 현재 문서에서 새 문서 주소를 할당해서 새 문서를 가져올 때
      • reload : 현재 문서를 다시 불러 올 때
      • replace : 현재 문서의 URL 을 지우고, 다른 URL의 문서로 교체. 히스토리가 삭제됨
      • toString : 현재 문서의 URL 을 문자열로 반환
  • screen 객체
    • 사용자 화면의 크기 정보
    • 프로퍼티
      • availHeight : UI(메뉴, 작업표시줄 등) 영역을 제외한 영역의 높이
      • availWidth : UI(메뉴, 작업표시줄 등) 영역을 제외한 영역의 너비
      • height : UI 영역 포함한 화면의 높이
      • width : UI 영역 포함한 화면의 너비
      • orientation : 화면의 현재 방향
    • 메서드
      • lockOrientation : 화면 방향 잠금 기능
      • unlockOrientation : 화면 방향 잠금 해제 기능
  • 새 창 제어
    • open() 메서드 이용
    • 팝업창
      • window.open() 메서드 이용
      • 기본형 window.open(경로, 이름, 속성)
    • 창 제어 속성
      • width : 창의 너비, px 단위
      • height : 창의 높이, px 단위
      • left : 웹 브라우저 왼쪽에서의 위치를 px 단위로 지정
      • top : 웹 브라우저 위쪽에서의 위치를 px 단위로 지정
    • 스크롤 이동
      • scrollTo() : 스크롤 위치를 특정 좌표로 이동
      • scrollBy() : 스크롤을 현재 위치에서 상대 위치로 이동
      • 기본형 window.scrollTo(x좌표, y좌표); window.scrollBy(x좌표, y좌표);
      • behavior 속성
        • scrollTo(), scrolloBy() 의 매개변수에 객체 리터럴을 전달할 때
        • smooth 값 : 스크롤이 휠로 이동하듯이 부드럽게 이동하게 함
