조건문

  • if 문
    • 기본 구문1 if 조건식
    • 기본 구문2 if 조건식 else
    • 기본 구문3 if 조건식 else if 조건식 else
    • 조건이 true 일 때 if 문이 실행 아니면 다음 구문
  • 삼항연산자
    • 기본 구문 -> 조건 ? 실행1 : 실행2;
    • 예) age < 20 ? alert("미성년자 입니다.") : alert("성인 입니다");
  • switch 문
    • 처리할 명령이 많을 때 편리
    • 조건은 case 문의 값과 일대일 대응
    • case 를 실행 후 switch 종료
    • 조건과 일치하는 case 가 없을 경우 default 실행
    • default 에는 break 문이 없음
    • 기본형 switch (조건) { case 값1: 실행문; break; case 값2: 실행문; break; ... default: 실행문; }
