표(table) 태그

  • table, tr, td 태그로 구성
  • rowspan : 줄단위 합치기
  • colspan : 컬럼단위 합치기
  • 표의 구조를 제목, 본문, 요약 부분으로 구성
  • thead, tbody, tfoot 태그를 사용
  • (시맨틱 태그 사용 시) 시각 장애인용 디바이스에서도 쉽게 이해 가능
  • 자바스크립트로 본문 부분만 스크롤 가능하게 구현 가능
  • col 과, colgroup 태그 : 열을 그룹화할 때 사용
  • scope : 제목을 나타내는 셀의 범위(row 단위, column 단위로 구분) 표시
