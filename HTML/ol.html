OL/UL/LI

  1. OL(Ordered List)
    • 각 항목 앞에 숫자가 붙어서 정렬
    • type 속성 : 순서 목록의 숫자 지정
      • 1: 숫자(default)
      • a: 소문자
      • A: 대문자
      • i: 로마소문자
      • I: 로마대문자
    • start 속성 : 목록의 시작 번호 수정
  2. UL(Unordered List)
  3. dl, dt, dd
    • 이름(제목), 값(설명) 형태로 구성되어 있는 목록
    • <dt> 태그 - 이름(제목) 지정
    • <dd> 태그 - 값(설명) 지정
    • 하나의 <dt> 에 여러 개의 <dd> 값을 가질 수 있음
    • 이름(제목) 1
      값(설명) 1
      값(설명) 1-1
      이름(제목) 2
      값(설명) 2
      값(설명) 2-1
